# Brotherhood Factions UI

Modern **Factions Interface UI** designed for game servers (RAGE:MP).
The interface displays factions, members, leaders, and staff with interactive animations and a clean responsive layout.

## Features

* 🎮 Modern faction cards UI
* 📊 Animated members capacity bars
* 👑 Automatic leader detection
* 📋 Staff list with ranks and online status
* 🧩 Expandable faction panel
* 🖱️ 3D hover tilt effect on cards
* 🔔 Custom notification system
* 📱 Basic responsive support

## Technologies

* HTML
* CSS
* JavaScript
* Font Awesome

## Structure

```
/images        → UI icons and faction images  
index.html     → Main interface  
style.css      → UI styling  
script.js      → Logic and animations
```
